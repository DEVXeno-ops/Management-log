## 🧑‍💻 Usage

Once the setup is complete, you can run the bot using the following command:

```bash
npm start
```

### Slash Commands

This bot provides the following slash commands:

- **/serverinfo**: Displays detailed information about the server.
- **/unban <userid>**: Unbans a user by their User ID.
- **/help**: Displays help information about available commands.

### Example

To unban a user with the ID `1234567890`:
```bash
/unban 1234567890
```
